# Book App

An interactive web application for browsing, managing, and searching books. Includes a standard version and an NSFW-filtered version for safe content viewing.

---

## 🚀 Features

- Browse a list of books with details (title + author + book cover)
- Search from over 30 million books available on the internetarchive
- "Recommended for you" showing similar books to the one the user is currently reading
- A friendly AI chatbot powered by openrouter's api to help the user with any questions they have regarding literature
- Two versions:
  - **Standard Version:** Full content
  - **NSFW-Filtered Version:** Hides NSFW content
- Responsive and user-friendly interface


## 🛠️ Tech Stack

- **Languages used:** HTML, CSS, JavaScript
- **APIs used:** Internet Archive, Imagga (only in NSFW-Filtered version), OpenRouter


## ⚡ Installation

1. Clone the repository
git clone https://github.com/rayanzinnar123-ui/book-app.git
